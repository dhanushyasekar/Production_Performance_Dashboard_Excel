# Excel Production Dashboard

## Project Overview

The Excel Production Dashboard is a data analytics project designed to analyze production performance, operational efficiency, and cost trends using Microsoft Excel.

This dashboard transforms raw production data into meaningful business insights through data cleaning, pivot tables, pivot charts, and interactive visualizations. 

## Dashboard Preview

![Dashboard Screenshot](/dashboard.png)

## Objective

The main objective of this project is to build an interactive dashboard that enables users to monitor production metrics and support data-driven decision-making.

## Key Analysis Performed

- Analyzed total production cost by product type
- Evaluated employee-wise task performance
- Tracked monthly and yearly production trends
- Calculated average production cost per unit
- Created visual reports for quick business insights

## Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Dashboard Design

## Skills Demonstrated

- Data Analysis
- Business Intelligence Reporting
- Data Visualization
- Excel Dashboard Development
- KPI Analysis

## Project Insights

- Identified high-performing product categories based on production output
- Compared cost variations across product types
- Analyzed employee productivity trends
- Observed production patterns over time

## Conclusion

This project demonstrates the ability to convert raw operational data into an interactive Excel dashboard that provides actionable insights for improving production efficiency and business performance.
